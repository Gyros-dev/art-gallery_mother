# Анна Векслер — онлайн-галерея

Сайт-портфолио художницы: гобелены ручного ткачества и текстильные коллажи.
Статический сайт (HTML/CSS/JS без сборки), работает на GitHub Pages.

Прокачанная версия исходного наброска: единый арт-дирекшен по палитре работ
(индиго, бирюза, терракота, золото, тёплый песок), анимации ткачества,
иммерсивная галерея-coverflow, лайтбокс, таймлайн выставок и адаптив.

## Страницы

| Файл | Раздел |
|---|---|
| `index.html` | Лендинг: хиро с анимацией «ткацкого станка», интро, лента избранного |
| `pages/gallery.html` | Галерея — coverflow с 3D-боковинами, фильтры, лента миниатюр, лайтбокс |
| `pages/exhibitions.html` | Выставки — таймлайн, превью сайта выставки в оверлее |
| `pages/literature.html` | Литература — публикации с просмотром PDF |
| `pages/about.html` | Художник — биография, статистика, контакты |

## Как добавить или изменить работы

Всё содержимое — в папке `data/` и `images/`. Код трогать не нужно.

**Работы галереи** — файл [`data/images.json`](data/images.json). Каждая работа:

```json
{
  "src": "images/art/1.jpg",
  "title": "Название работы",
  "category": "гобелен",
  "year": "2020",
  "material": "Шерсть, ручное ткачество",
  "size": "50 × 70 см",
  "location": "Санкт-Петербург"
}
```

- `src` — путь к файлу от корня сайта (положите изображение в `images/art/`).
- `category` — `гобелен` или `коллаж`; из категорий автоматически строятся фильтры.
- Порядок работ в галерее = порядок в JSON.

**Выставки** — [`data/exhibitions.json`](data/exhibitions.json). Поле `status`:
`upcoming` (текущие/будущие, с меткой «Скоро») или `past` (архив).
Если указан `url`, карточка открывает сайт выставки в оверлее.

**Литература** — список в [`pages/literature.html`](pages/literature.html); PDF кладите в `docs/`.

> ⚠️ **Названия работ №2–8 — черновые заглушки**, проставленные по сюжету
> изображений. Замените их настоящими названиями в `data/images.json`.
> Контактный e-mail в `pages/about.html` тоже пример — впишите реальный.
> Файлы `images/art/papa_1.jpg` и `papa_2.jpg` в галерею не включены —
> если это работы Анны, добавьте их в `data/images.json`.

## Локальный просмотр

```bash
cd art-gallery_mother
python3 -m http.server 4188
# открыть http://localhost:4188
```

## Публикация на GitHub Pages

Репозиторий уже опубликован на `gyros-dev.github.io/art-gallery_mother`.
Чтобы обновить сайт этой версией:

```bash
git add -A
git commit -m "Прокачанная версия галереи: арт-дирекшен и анимации"
git push
```

В настройках репозитория: **Settings → Pages → Source → GitHub Actions**
(workflow [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml) публикует
сайт при каждом пуше). Сборка не нужна — сайт статический.

## Структура

```
index.html            — лендинг
pages/                — внутренние страницы + navbar.html (общая шапка)
css/                  — base.css (дизайн-система) + по стилю на страницу
js/
  common.js           — навбар, зерно, появления, канвас-«станок»
  gallery.js          — coverflow, фильтры, лайтбокс
  exhibitions.js      — таймлайн выставок
data/                 — images.json, exhibitions.json
images/, docs/        — изображения работ, портрет, PDF
```
